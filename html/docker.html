Docker

  • Docker — платформа для управления контейнерами, которая позволяет легко создавать, развертывать и запускать приложения в изолированных средах. Основная задача Docker — запустить приложение и при этом не тратить много времени на настройку.
  • Docker Image — набор приложений, библиотек со всеми зависимостями и настройками, из которого разворачивается контейнер. Docker Image создается на основе Docker-файла, в котором указывается всё необходимое для работы приложения.
  • Docker Container — работающее приложение, созданное на базе Docker Image.
