Git

  • Git — это быстрая, масштабируемая, распределённая система управления версиями с необычно богатый набор команд, обеспечивающий обе операции на высоком уровне и полный доступ к внутренним устройствам
  • init - команда здания создания нового репозитария на локальной машине.
  • branch - команда применяется для отображения, создавния или удаление ветки.
  • push - команда выгружает изменения в удаленный репозитарий.
  • pull - команда для получения изменений из удаленного репозитария на локальную машину (репозитарий).
